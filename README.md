# SoulHub Frontend (Vite + React + TS)

## Quick Start
```bash
npm i
npm run dev
```

## Pages
- `/` Home
- `/explore`
- `/character/:id`
- `/commit/:id`
- `/verify/:id`
- `/create`
- `/dashboard`
- `/about`

## Notes
- Includes TailwindCSS
- Includes Monaco-based JSON editor
- Components: SafetySeal, IngredientTable, CommitTimeline, etc.
