# SoulHub Frontend (Vite + React + TS)

## Quick Start
```bash
npm i
npm run dev
```

## Pages
- `/` Home
- `/explore`
- `/character/:id`
- `/commit/:id`
- `/verify/:id`
- `/create`
- `/dashboard`
- `/about`
